# Slider / RangeSlider

A custom (range) slider made to suit everyone's needs. Meant to replace Gutenberg's `RangeControl` while being less prone to bugs from WP updates and with more (and more stable) features.

It's based on [`rc-slider`](https://slider-react-component.vercel.app/), most of the default props apply here, with some additions.

## Basic slider

```jsx
const [sliderValue, setSliderValue] = useState(1);

<Slider
	label={__('Basic slider', 'eightshift-frontend-libs')}
	value={sliderValue}
	onChange={(value) => setSliderValue(value)}
	min={1}
	max={10}
/>
```

### Basic range slider
```jsx
const [sliderRangeValue, setSliderRangeValue] = useState([1, 2]);

<Slider
	label={__('Basic slider', 'eightshift-frontend-libs')}
	value={sliderRangeValue}
	onChange={(value) => setSliderRangeValue(value)}
	min={1}
	max={10}
/>
```
