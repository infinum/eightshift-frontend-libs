# LinkEditComponent

A inline URL editor that looks nice and fits into the other Gutenberg and Eightshift controls.

## Usage

To use it:
1. Import the component from Frontend Libs
2. Add the `LinkEditComponent` into the toolbar

```jsx
import { LinkEditComponent } from '@eightshift/frontend-libs/scripts/components';
import { getAttrKey } from '@eightshift/frontend-libs/scripts/helpers';

// ...

export const ButtonOptions = (attributes) =>  {
	return (
			// ...

			<LinkEditComponent
				url={buttonUrl}
				opensInNewTab={buttonOpensInNewTab}
				setAttributes={setAttributes}
				anchorRef={ref}
				title={label}
				urlAttrName={getAttrKey('buttonUrl', attributes, manifest)}
				isNewTabAttrName={getAttrKey('buttonOpensInNewTab', attributes, manifest)}
			/>
	)
}
```
