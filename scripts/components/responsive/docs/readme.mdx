# Responsive

A component to provide options that can change across breakpoints. Provides optional 'Inherit' button to handle inheritance across the breakpoints.

## Example

```jsx
<Responsive label={__('Demo option', 'eightshift-frontend-libs'}>
	// Child items go here, usually:
	{options.breakpoints.map((breakpoint, index) => {
		const point = ucfirst(breakpoint);
		const attr = `myAttributeName${point}`;

		return (
			<Fragment key={index}>
				// ...
			</Fragment>
		);
	})}
</Responsive>
```
