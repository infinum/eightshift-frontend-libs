# Repeater

A simple repeater that allows showing a list of complex options in a nice way.

## Usage

To use it:
1. Import `Repeater` and `RepeaterItem` from Frontend Libs
2. Add the `Repeater` into the options

```jsx
import { getAttrKey } from '@eightshift/frontend-libs/scripts/helpers';

// ...

export const VideoOptions = (attributes) =>  {
	const {
		setAttributes,
	} = attributes;

	const videoCaptions = checkAttr('videoCaptions', attributes, manifest);

	return (
			// ...

			<Repeater
				icon={icons.videoSubtitleAlt}
				label={__('Captions', 'eightshift-frontend-libs')}
				items={videoCaptions}
				attributeName={getAttrKey('videoCaptions', attributes, manifest)}
				setAttributes={setAttributes}
			>
				{videoCaptions.map((item) => (
					<RepeaterItem
						key={item.id} // id is auto-generated
						icon={item.icon} // you'll probably have a function to return an icon based on type
						title={item.title}
					>
						// Your options here
					</RepeaterItem>
				))}
			</Repeater>
	)
}
```
