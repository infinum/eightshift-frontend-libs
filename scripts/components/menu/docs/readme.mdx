# Menu

A simple dropdown menu.

## Usage

```jsx
<Menu label={__('File', 'eightshift-frontend-libs')}>
	<MenuItem label='Add item' icon={icons.add} />
	<MenuItem label='Demo item 1' icon={icons.num1Circle} />
	<MenuItem label='Demo item 2' icon={icons.num2Circle} />
	<MenuSeparator />
	<MenuItem label='Remove item' icon={icons.remove} />
</Menu>
```
