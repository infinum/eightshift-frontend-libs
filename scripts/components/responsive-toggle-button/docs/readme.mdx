# ResponsiveToggleButton

A simple toggle button that allows picking values through breakpoints.

## Usage

```jsx
<ResponsiveToggleButton
	icon={icons.emptyRect}
	label='Demo control'
	value={value}
	onChange={(v) => setAttributes(v)}
/>
```
