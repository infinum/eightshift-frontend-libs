# NumberPicker

A simple component that allows picking numbers, can replace a Slider.

## Usage

```jsx
<NumberPicker
	min={1}
	max={10}
	value={myAttribute}
	onChange={(value) => setAttributes({ myAttribute: value })}
/>
```
