# OptionSelector

A simple component for picking mutually exclusive options.

## Usage

```jsx
<OptionSelector
	label={__('Size', 'eightshift-frontend-libs')}
	options={options}
	onChange={(val) => setAttributes(...)}
	value={value}
/>
```

Each option should be in this format:

```js
{
	icon: icons.large, // Optional, can be either ES icon (name or whole icon element) or dashicon name.
	value: 'l',
	label: __('L', 'eightshift-frontend-libs'), // Optional
	tooltip: __('Large', 'eightshift-frontend-libs'), // Optional, label will be used as fallback
}
```
