# AnimatedContentVisibility

A wrapper around the Gutenberg `Animated` component, to make our components less boilerplate-y.

## Usage

```jsx
<AnimatedContentVisibility showIf={myVariable}>
	Content goes here.
</AnimatedContentVisibility>
```
