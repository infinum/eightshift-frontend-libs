# ResponsiveSlider

A `Slider` that allows changing values between breakpoints.

## Usage

```jsx
<ResponsiveSlider
	value={value}
	onChange={(value) => setValue(value)}
	min={0}
	max={50}
	step={1}
	noBottomSpacing={true}
	icon={icons.emptyRect}
	label='Option name'
/>
```
