# Toggle

A toggle/checkbox control with an optional icon.

## Usage

```jsx
<Toggle
	icon={icons.width}
	label={__('Wide button', 'eightshift-frontend-libs')}
	checked={buttonFullWidth}
	onChange={(value) => setAttributes({ [getAttrKey('buttonFullWidth', attributes, manifest)]: value })}
/>
```
