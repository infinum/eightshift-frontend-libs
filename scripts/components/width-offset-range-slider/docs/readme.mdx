# WidthOffsetRangeSlider

A wrapper component for picking column width & offset in a nice, user-friendly way.

## Usage

```jsx
<WidthOffsetRangeSlider
	offsetAttributeName='componentOffset'
	widthAttributeName='componentWidth'
	isFullWidthAttributeName='componentIsFullWidth' // (optional)
	manifest={manifest}
	attributes={attributes}
	setAttributes={setAttributes}
	showFullWidthToggle // (optional, default true)
	includeGutters // (optional, default true)
	showOffsetAutoToggle // (optional, default false)
/>
```
