# ToolbarOptionPicker

A flexible control that allows nice-looking radio-like option selection with multiple modes.

It can be used as:
- a toolbar button that opens a dropdown on click (with either the icon of currently selected item or custom icon)
- an inline toolbar button group
- standalone button that opens a dropdown on click (with either the icon of currently selected item or custom icon)

## Properties
| Property | Description |
|---|---|
| `value` | Current value |
| `onChange` | Callback that gets called when an option is changed |
| `label` | Label that gets shown when hovering over the toggle icon (doesn't apply to inline) |
| `screenReaderDescription` | _(optional)_ How the component gets described to a screen reader |
| `isInToolbar` | If `true`, the component is rendered as a `ToolbarGroup`, if false it's rendered as a `DropdownMenu` |
| `isToolbarButton` | If `true`, the components is rendered as a Toolbar button, otherwise it's rendered as a regular button |
| `isInline` | If used in conjunction with `isInToolbar` renders the options inline instead of a button showing a dropdown menu |
| `popoverPosition` | _(optional)_ Sets the location of the popover that shows on click. Doesn't apply when `isInline` is set. Default is `bottom right`. |
| `buttonIcon` | _(optional)_ If set and the picker is not inline, the button that triggers the dropdown will have the icon defined in `buttonIcon` shown instead of the icon of the currently active option. |
| `isToggleButtonActive` | _(optional)_ If true, the trigger button has an _active_ style. Default is `false`. |
| `showToggleButtonLabel` | _(optional)_ If true, the text label is shown next to the icon of the toggle button. Default is `false`. |

## Usage examples
### Toolbar button

```jsx
import { ToolbarOptionPicker } from '@eightshift/frontend-libs/scripts/components';

// ...

export const ButtonToolbar = (attributes) =>  {
	return (
			// ...

			<ToolbarOptionPicker
				value={buttonAlign}
				onChange={(value) => setAttributes({ [`${componentName}Align`]: value })}
				isToolbarButton={true}
				isInToolbar={true}
				options={getOptions(manifest, componentName, 'align', options)}
				description={__('Change text alignment', 'eightshift-frontend-libs')}
				label={__('Align', 'eightshift-frontend-libs')}
			/>
	)
}
```

### Standalone button

```jsx
import { ToolbarOptionPicker } from '@eightshift/frontend-libs/scripts/components';

// ...

export const ButtonOptions = (attributes) =>  {
	return (
			// ...
```
