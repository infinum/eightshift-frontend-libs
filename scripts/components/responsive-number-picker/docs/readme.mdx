# ResponsiveNumberPicker

A simple wrapper for NumberPicker that allows picking values through breakpoints.

## Usage

```jsx
<ResponsiveNumberPicker
	icon={icons.emptyRect}
	label='Demo control'
	value={value}
	onChange={(v) => setAttributes(v)}
	min={0}
	max={100}
/>
```
