# Lists Block

Block used to provide lists functionality to your project.

## Dependencies

1. components/lists

## Implementation

Open a terminal and type in this command inside your projects root:

```shell
wp boilerplate blocks use-block --name=lists
wp boilerplate blocks use-component --name=lists
```
