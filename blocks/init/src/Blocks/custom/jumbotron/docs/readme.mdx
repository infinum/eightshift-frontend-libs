# Jumbotron Block

Block used to provide jumbotron functionality to your project.

## Dependencies

1. components/jumbotron

## Implementation

Open a terminal and type in this command inside your projects root:

```shell
wp boilerplate blocks use-block --name=jumbotron
wp boilerplate blocks use-component --name=jumbotron
```
