# Video Block

Block used to provide video functionality to your project.

## Dependencies

1. components/video

## Implementation

Open a terminal and type in this command inside your projects root:

```shell
wp boilerplate blocks use-block --name=video
wp boilerplate blocks use-component --name=video
```
