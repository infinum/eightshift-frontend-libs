# Button Block

Block used to provide button functionality to your project.

## Dependencies

1. components/button

## Implementation

Open a terminal and type in this command inside your projects root:

```shell
wp boilerplate blocks use-block --name=button
wp boilerplate blocks use-component --name=button
```
