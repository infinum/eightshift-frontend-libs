# Social networks Block

Block used to provide links to social networks or an ability to share the current page on social networks.

## Dependencies

1. components/social-networks

## Implementation

Open a terminal and type in this command inside your projects root:

```shell
wp boilerplate blocks use-component --name=social-networks
wp boilerplate blocks use-block --name=social-networks
```
