# Image Block

Block used to provide image functionality to your project.

## Dependencies

1. components/image

## Implementation

Open a terminal and type in this command inside your projects root:

```shell
wp boilerplate blocks use-component --name=image
wp boilerplate blocks use-block --name=image
```
