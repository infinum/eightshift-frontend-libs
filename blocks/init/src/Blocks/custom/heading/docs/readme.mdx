# Heading Block

Block used to provide heading functionality to your project.

## Dependencies

1. components/heading

## Implementation

Open a terminal and type in this command inside your projects root:

```shell
wp boilerplate blocks use-block --name=heading
wp boilerplate blocks use-component --name=heading
```
