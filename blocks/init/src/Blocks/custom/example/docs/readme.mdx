# Example Block

Block used to provide an example of blocks structure and functionality.

## Implementation

Open a terminal and type in this command inside your projects root:

```shell
wp boilerplate blocks use-block --name=example
```

## Note

This block doesn't provide any specific functionality because it is only a boilerplate for your new block.
